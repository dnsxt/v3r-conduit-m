Purpose: NEXUS routing utility. Shared module imported by all handlers.
Exposes route_llm(route_id, system_prompt, user_prompt, max_tokens=1024,
temperature=0.3).

Key functions:
- get_route(route_id): GetItem on v3r-routing
- get_key(provider): GetItem on v3r-keys
- reset_if_new_day(provider, key_record, today): lightweight daily reset
- increment_usage(provider): UpdateItem requests_today + 1
- call_openai_compat(key_record, ...): HTTPSConnection POST to
  /v1/chat/completions
- route_llm(...): main entry — resolves route, iterates failover chain,
  returns normalized text

Design decisions:
- FP-002 compliance: http.client.HTTPSConnection exclusively
- base_url stored without https:// prefix — host only per HTTPSConnection
- daily_limit defaults to 9999 if field absent — safe default
- status=exhausted set on key when daily_limit reached
- All exceptions caught per provider — chain continues on any failure
- Unknown route_id falls back to 'default' before hard fail
